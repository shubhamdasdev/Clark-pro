# Combined Weekly Cadence

Both workstreams run side by side. This is the unified weekly schedule, blending the short-form pipeline and long-form publishing.

## At a Glance

- **Short-form:** ~7 reels + 7 carousels per week (1 of each per day).
- **Long-form:** 3 LinkedIn posts + 1 newsletter + (1 Medium per month).
- **Real footage:** 2 lifestyle clips/week from a Friday night out.
- **Daily human time:** 60–90 min weekdays, ~2 hrs Saturday, 30 min Sunday.

## Weekly Schedule

| Day | AM (~30 min) | PM (~30–60 min) |
|---|---|---|
| **Mon** | LinkedIn post (insight or build update) | Generate next 7 reel scripts in Claude Code (batch); post 1 reel + 1 carousel |
| **Tue** | Daily note dump (5 min) | Voiceover batch + start avatar render queue; post 1 reel + 1 carousel |
| **Wed** | LinkedIn post (AI advancement decoded) | Avatar renders complete; edit 3 reels; post 1 reel + 1 carousel |
| **Thu** | Daily note dump | Edit 4 more reels; generate 7 carousels with `make_carousel.py`; post 1 reel + 1 carousel |
| **Fri** | LinkedIn post (Cortex / personal milestone) | Real-footage capture (night out, bar, outfit); post 1 reel + 1 carousel |
| **Sat** | **Newsletter draft + send via Beehiiv** (~90 min) | Cut real footage into 2 lifestyle reels; post 1 reel + 1 carousel |
| **Sun** | Plan next week's hooks (refill bank) | Weekly review (30 min): retention numbers + format kill list. Last Sunday/month: ship Medium piece. |

## Posting Times

- **TikTok:** 7–9pm CT
- **IG Reels & Carousels:** 8–10pm CT
- **LinkedIn:** 8–10am CT (Mon, Wed, Fri)
- **Newsletter:** 9am CT Saturday
- **Medium:** Sunday evening (last Sunday of month)

Adjust after 3 weeks of analytics.

## Time Budget Sanity Check

| Activity | Weekly time |
|---|---|
| Short-form pipeline | ~6 hrs |
| Long-form (LinkedIn 3× + newsletter + monthly Medium amortized) | ~3 hrs |
| Real-footage capture + edit | ~2 hrs |
| Weekly review + planning | ~30 min |
| **Total** | **~11–12 hrs/week** |

That's ~1.5 hrs/day average — within the budget. Saturday and Sunday absorb the heavier work; weekday evenings are 30–60 min.

## Daily Note Dump (the long-form fuel)

5 minutes/day. Capture in Apple Notes or Obsidian:
- Anything interesting that happened at work (patterns only, never proprietary)
- One AI/ML thing you read and a take on it
- One Cortex decision and its trade-off
- One thing about Chicago / dating / lifestyle worth writing about

Saturday's newsletter draft pulls from these notes. Without the daily dump, Saturday becomes painful.

## When the Schedule Breaks

If a day slips:
1. **Skip a day, never a week.** The system survives a missed Tuesday — it doesn't survive a missed Saturday newsletter for two weeks running.
2. **Cut quality on the lowest-stakes channel first** (a phoned-in TikTok). Don't cut newsletter or LinkedIn — those are owned audience.
3. **If 3 days slip in a row,** pause non-essentials and ship just: 1 LinkedIn post + 1 reel that day. Get the streak back.

## Sundays Are Sacred

The 30-min weekly review is the most important block in the schedule. Skip it for a few weeks and you'll notice the content quality drift downward without knowing why. Always:
1. Top 3 + bottom 3 short-form videos by retention.
2. Note the hooks.
3. Pick 1 format/hook to repeat next week.
4. Kill any format with <30% avg retention after 3 attempts.
